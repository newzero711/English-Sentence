# 영어 문장 암기 앱 — 설치 & 사용 가이드

## 1. Google Sheets 준비

**탭(워크시트) 하나가 단어장 하나**입니다. 탭 이름이 곧 단어장 이름으로 앱에 표시됩니다. 단어장을 늘리고 싶으면 시트 하단에 탭을 새로 추가하고 원하는 단어장 이름으로 바꿔주세요 (예: "일상회화", "비즈니스").

각 탭의 헤더(1행)에는 아래 컬럼이 있으면 됩니다 (순서는 상관없음, 헤더 텍스트로 자동 인식).

| 날짜 | 표현(영어) | 내 대사(한글) | Details (부가설명) |
|------|-----------|--------------|---------------------|
| 2. 5 | that's how the cookie crumbles, right? | 어쩔수 없지 | 세상이 다 그런 거지. / 그게 인생이지. |

- **표현(영어)**: 외울 영어 표현 — 정답 채점 기준이 됩니다.
- **내 대사(한글)**: 학습 화면에 먼저 보여주는 한글 문장입니다.
- **Details (부가설명)**: 선택사항. 입력해두면 학습 화면에서 "힌트 보기"로 펼쳐볼 수 있습니다.
- 헤더 텍스트는 "표현", "영어", "대사", "한글", "Details", "부가", "설명", "날짜" 키워드를 포함하면 자동으로 인식됩니다.
- 표현/영어, 대사/한글 컬럼이 모두 인식되지 않는 탭은 단어장이 아닌 것으로 보고 동기화 시 건너뜁니다.

**`학습로그`** 라는 이름의 탭은 예약되어 있어 단어장으로 취급되지 않습니다 — 동기화 시 Apps Script가 학습 기록용으로 자동 생성/사용합니다.

## 2. Apps Script 배포

1. Sheets 메뉴에서 **확장 프로그램 → Apps Script** 클릭
2. 기본 `Code.gs` 내용을 전부 지우고, `AppsScript_Code.gs` 파일 내용을 그대로 붙여넣기
3. 우측 상단 **배포 → 새 배포**
   - 유형 선택: **웹 앱**
   - 실행 사용자: **나**
   - 액세스 권한: **모든 사용자** (또는 "Google 계정이 있는 모든 사용자")
4. 배포 후 생성되는 웹앱 URL (`https://script.google.com/macros/s/.../exec`) 을 복사

> 시트 내용을 수정한 뒤에는 "배포 → 배포 관리 → 수정 → 새 버전"으로 재배포해야 변경사항이 반영됩니다.

## 3. 아이폰에 앱 설치 (PWA)

이 폴더의 파일들(`index.html`, `manifest.json`, `sw.js`, 아이콘)을 정적 호스팅이 가능한 곳에 올려야 합니다. 가장 쉬운 방법:

### 옵션 A — GitHub Pages (무료, 추천)
1. GitHub에 새 저장소 생성, 이 파일들을 업로드
2. 저장소 설정 → Pages → main 브랜치로 활성화
3. 생성된 URL (`https://username.github.io/repo명/`)을 아이폰 Safari로 접속

### 옵션 B — 다른 정적 호스팅
Netlify, Vercel, Cloudflare Pages 등에 폴더를 드래그&드롭하면 즉시 URL이 생성됩니다.

### 홈 화면에 추가
1. 아이폰 **Safari**에서 배포된 URL 접속 (Chrome은 PWA 설치 미지원)
2. 하단 **공유 버튼** → **홈 화면에 추가**
3. 이름 확인 후 **추가** → 홈 화면에 앱 아이콘 생성

## 4. 앱에서 첫 설정

1. 앱 실행 → 하단 **⚙ 설정** 탭
2. **Apps Script 웹앱 URL** 칸에 2단계에서 복사한 URL 붙여넣기
3. **지금 동기화** 버튼 클릭 → 시트 내용을 불러옵니다

## 5. 화면 구성 (하단 탭)

- **📚 단어장**: 전체/단어장별 문장 목록. 목록 위 **학습 시작** 버튼을 누르면 그 단어장의 문장으로 학습 카드가 시작됩니다. 각 문장의 암기 상태(안 외움/암기완료/다시 외우기)가 표시되고, 힌트가 있으면 펼쳐볼 수 있습니다.
- **➕ 문장추가**: 단어장/영어 표현/한글 대사/힌트를 입력해 새 문장을 추가합니다. 시트 URL이 설정되어 있으면 즉시 시트에도 한 줄 추가됩니다(오프라인이면 대기열에 쌓여뒀다가 다음 동기화 때 반영). 입력한 단어장 이름의 탭이 시트에 없으면 Apps Script가 자동으로 새 탭을 만들어줍니다.
- **📅 달력**: 날짜를 누르면 그날 단어장별로 몇 문장을 학습했는지, 정답률이 얼마였는지 보여줍니다. 이 기록은 기기에 저장되며 학습 기록 초기화 시 함께 지워집니다.
- **⚙ 설정**: 시트 URL/동기화, 유사도 채점 기준(50~100%, 기본 85%), 학습 기록 CSV 내보내기, 초기화.

### 학습 카드 화면

- 한글 대사를 보고 영어로 작문 → **채점하기**
- 유사도 기반 채점: 대소문자, 마지막 문장부호, 단순 복수형(s) 차이는 무시
- 상단 필터 칩: **전체 / 오답만 / 안 푼 문장** 으로 복습 범위 선택 (단어장 탭에서 선택한 단어장 범위 내에서 동작)
- 학습 기록은 기기에 우선 저장되고, 동기화 시 시트의 `학습로그` 탭에 일괄 업로드
- 오프라인에서도 캐시된 문장 목록으로 학습 가능, 인터넷 연결 시 동기화

## 파일 구성

```
sentence-app/
├── index.html          # 앱 본체 (UI + 로직)
├── manifest.json        # PWA 설정 (홈화면 아이콘/이름)
├── sw.js                 # 서비스 워커 (오프라인 캐시)
├── icon-192.png          # 앱 아이콘
├── icon-512.png          # 앱 아이콘 (고해상도)
└── AppsScript_Code.gs    # Google Sheets 연동 스크립트
```
