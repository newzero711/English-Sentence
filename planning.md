# 영어 문장 암기 앱 — 기획

## 1. 기본 컨셉
- 사용자가 Google Sheets에 외우고 싶은 영어 문장을 직접 기록해오던 습관을 그대로 활용
- PWA(아이폰 홈화면 설치형 웹앱)로 구현, Apps Script를 통해 Google Sheets와 데이터 연동(불러오기/내보내기)

## 2. 데이터 소스 (Google Sheets)
- **탭(워크시트) 하나 = 단어장 하나**. `학습로그` 탭을 제외한 모든 탭을 읽어서 통합하며, **탭 이름이 곧 단어장 이름**
- 새 단어장을 만들려면 시트 하단에 탭을 새로 추가하면 됨 (탭 이름 = 단어장 이름)
- 각 탭의 컬럼 (순서 무관, 헤더 텍스트로 자동 인식):
  - **날짜**
  - **표현(영어)** — 암기 대상, 채점 기준 답안
  - **내 대사(한글)** — 학습 화면에 먼저 보여주는 프롬프트
  - **Details(부가설명)** — 선택, 힌트로 펼쳐보기
- 표현/영어 또는 대사/한글 컬럼이 인식되지 않는 탭은 단어장이 아닌 것으로 보고 동기화 시 건너뜀
- `학습로그` 탭은 동기화 시 Apps Script가 자동 생성 (timestamp/sentenceId/단어장/사용자입력/정답여부/유사도)

## 3. 학습 방식
- 한글 대사를 보고 영어 표현을 작문 → 채점
- Levenshtein 유사도 기반 채점 (대소문자/마지막 문장부호/단순 복수형 차이는 허용)
- 유사도 기준 50~100% 조절 가능 (기본 85%)
- 필터: 전체 / 오답만 / 안 푼 문장

## 4. 홈화면 구조 (하단 탭 4개)

| 탭 | 기능 |
|---|---|
| 📚 단어장 | 단어장별 칩 필터 + 전체 문장 리스트(한글/영어/힌트), 암기 상태 표시(안 외움/암기완료/다시 외우기), **학습 시작** 버튼으로 학습 카드 진입 |
| ➕ 문장추가 | 단어장/영어/한글/힌트 입력 → 로컬 즉시 반영 + 시트에도 동기화(오프라인 시 대기열). 새 단어장 이름을 입력하면 시트에 새 탭이 자동 생성됨 |
| 📅 달력 | 월별 그리드, 학습한 날 점 표시, 날짜 탭 시 그날 단어장별 문장 수 + 정답률 |
| ⚙ 설정 | 시트 URL/동기화, 유사도 기준, CSV 내보내기, 학습 기록 초기화 |

학습 카드 화면은 단어장 탭에서 "학습 시작"을 눌렀을 때 뜨는 전체화면 오버레이로 분리됨.

## 5. 결정했던 주요 선택지
- 단어장 구분: **시트 탭(워크시트) 단위**로 채택 — 탭 이름 = 단어장 이름. 새 단어장은 시트 하단에 탭을 추가하면 됨 (이전에는 "단어장" 컬럼 하나로 구분했으나, 탭으로 직접 분리하는 흐름이 더 자연스럽다고 판단해 변경)
- 문장추가 저장 위치: 로컬에만 두지 않고 **Google 시트에도 즉시 동기화**
- 달력 상세정보: 그날 **단어장별 문장 수 + 정답률**로 표시 (전체 합산이 아니라 단어장별 분리)
- 학습 흐름: 한글 → 영어 (영어 → 한글 방향은 채택 안 함)

## 6. 기술 메모
- 로컬 저장(localStorage): 문장 목록, 학습 로그, 동기화 대기 로그/문장, 달력용 히스토리, 설정 — 모두 별도 키로 분리 저장
- 달력 기록은 **기기 로컬 전용** (여러 기기 간 동기화는 지원 안 함, 학습 기록 초기화 시 함께 삭제됨)
- Apps Script는 헤더 텍스트 키워드 매칭 방식으로 컬럼을 인식해서, 사용자가 컬럼 순서를 바꿔도 깨지지 않음
